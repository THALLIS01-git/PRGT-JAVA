# ☕ Meus Estudos de Java

Bem-vindo ao meu repositório de estudos e consultas de Java! Este espaço foi criado para centralizar meus aprendizados, revisar conceitos essenciais, relembrar a sintaxe da linguagem e guardar trechos de código úteis para reaproveitar em projetos futuros.

---

## 📌 Objetivo do Repositório

* **Fixação de Conceitos:** Guardar anotações sobre lógica de programação, Programação Orientada a Objetos (POO) e estruturas de dados.
* **Guia de Sintaxe:** Servir como um "cheat sheet" rápido para quando eu esquecer como funciona uma estrutura específica.
* **Banco de Código:** Armazenar códigos prontos e mini-projetos que posso evoluir com o tempo.

---

## 📂 Estrutura do Repositório

A organização está um pouco bagunçada por conta dos meus raciocínios e evoluções! Mas, no geral, você vai encontrar os códigos espalhados entre conceitos básicos de sintaxe, estudos de Programação Orientada a Objetos (POO), estruturas de dados e alguns mini-projetos que vou desenvolvendo e continuando com o tempo.
